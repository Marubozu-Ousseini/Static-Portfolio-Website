Credly API Test Tool

This tool helps you test your Credly profile and troubleshoot API issues.

Instructions

  1. Make sure your Credly profile is public
  2. Enter your Credly User ID (found in your profile URL)
  3. Click "Test API" to check if your badges can be fetched
  4. If successful, you'll see your badges listed below
  5. If it fails, you can use manual certifications as a fallback

Your Credly Profile URL should be:

https://www.credly.com/users/YOUR_USER_ID/badges

API Endpoint being tested:

https://www.credly.com/users/YOUR_USER_ID/badges.json
